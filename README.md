## 🚀 What is this App?

**Player Pressure Analysis & FitnessForge** is a next-generation web application that blends real-time cricket match analytics with personalized fitness recommendations, powered by AI and Google Gemini. Whether you're a cricket fan, player, coach, or fitness enthusiast, this app gives you:

- **Live Pressure Analysis:** See how match context (score, wickets, crowd, venue) impacts player pressure in real time, with narrative insights for both batters and bowlers.
- **FitnessForge:** Compare your own fitness profile to legendary cricket players, get AI-driven feedback, and discover how to train like the pros.
- **Interactive Chat & Insights:** Engage with live match data, ask questions, and receive instant, AI-generated responses.
- **Modern, Responsive UI:** Built with React, Tailwind CSS, and Lucide icons for a beautiful, seamless experience.

---

## 🌟 Why is it Useful?

- **For Fans:** Deepen your understanding of the game with pressure analytics and behind-the-scenes context.
- **For Players & Coaches:** Get actionable insights to improve performance under pressure and tailor fitness routines.
- **For Everyone:** Experience the power of AI in sports analytics and personal health, all in one place.

## 🏏 Key Features

- **Cricket Match Pressure Meter:** Calculates and visualizes pressure based on live match data.
- **AI-Powered Fitness Comparison:** Upload your profile and see how you stack up against cricket legends.
- **Real-Time Chat & Analysis:** Powered by Google Gemini and Firebase.
- **Socket.IO Integration:** For live updates and chat.
- **Firebase Auth & Firestore:** Secure, real-time data storage and authentication.

## 🛠️ Getting Started

**Prerequisites:** Node.js, Google Gemini API Key

1. Clone the repo and install dependencies:
   ```bash
   npm install
   ```
2. Set your Gemini API key in `.env.local`:
   ```env
   GEMINI_API_KEY=your-gemini-api-key
   ```
3. Run the app locally:
   ```bash
   npm run dev
   ```
4. Open [http://localhost:5173](http://localhost:5173) in your browser.

---

## ☁️ Deploying to Google Cloud Platform (GCP)

1. Build the app:
   ```bash
   npm run build
   ```
2. Deploy using Google Cloud Build or App Engine:
   ```bash
   gcloud app deploy
   ```
